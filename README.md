# Shadowrocket config + auto-updating rules

## Install (Stable)
Config → + → Download from URL:

https://cdn.jsdelivr.net/gh/vasyakin/shadowrocket-rules@main/config/setup_stable.conf

## Install (Aggressive)
https://cdn.jsdelivr.net/gh/vasyakin/shadowrocket-rules@main/config/setup_aggressive.conf

## Updating rules
Rules are loaded via RULE-SET from /rules/** and update automatically after you edit files in this repo.
